## how to test ?

add remote or download source then :


```bash
flutter pub get
```
```bash
flutter pub upgrade
```
```bash
flutter run
```
<hr>


used package in Dashboard Ui :

* GoogleFonts: `google_fonts: ^6.2.1`
* IconSax: `iconsax_flutter: ^1.0.1`
* FlChart: `fl_chart: ^1.0.0`
